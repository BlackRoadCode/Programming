Usefull Commands

Comando

Descripción

git reflog

 Te va a mostrar todos los commit que realizaste sin importar si los borraste o no.

git log -3 --name-only --pretty=format:''

 Lista de archivos modificados en n confirmaciones previas

git rm -r --cached .

 Eliminar el caché de GIT, sirve por ejemplo cuando el .gitignore no funciona (se sigue tomando la versión cacheada).

git log --oneline

 print out the commit history of the repository in one line.
